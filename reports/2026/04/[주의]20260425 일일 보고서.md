- **CVE ID**: CVE-2026-41080
- **CVSS 점수**: 미등록 (신규, 평가 진행 중) — 해시 플러딩 DoS 특성상 Medium~High 예상
- **영향 범위**: libexpat < 2.8.0
- **자사 영향**: **확인 필요 (UNKNOWN)** — `current_version: "확인 필요"` [버전 정보 미등록]
- **공식 링크**: https://github.com/libexpat/libexpat/releases/tag/R_2_8_0
- **발견 채널**: GitHub Releases (2026-04-24T10:02:54Z 배포)
- **취약점 요약**: 기존 해시 플러딩 방어(SipHash)가 16바이트 중 4~8바이트 엔트로피만 사용, 공격자가 크래프트된 XML 문서로 해시 충돌 유발 DoS 가능. 2.8.0에서 16바이트 전체 엔트로피 사용으로 수정됨. `XML_SetHashSalt` API 자체가 취약점으로 간주되어 deprecated 처리됨.
- **권장 조치**:
  - 업그레이드 대상 버전: **2.8.0**
  - XML_SetHashSalt() 호출 코드가 있다면 XML_SetHashSalt16Bytes() 또는 Expat 내부 자동 처리로 마이그레이션 필요
  - 참고: libexpat 프로젝트 인력 부족 공지 중 (미해결 보안이슈: https://github.com/libexpat/libexpat/issues/1160)
