## 과거 보고서 검색

```bash
# 특정 라이브러리 관련 보고서 찾기
grep -r "OpenSSL" reports/

# 긴급 보고서만 보기
find reports/ -name "[긴급]*"

# 최근 1주일 긴급 건 요약
git log --since="1 week ago" --grep="📊" -- reports/
```

## 보고서가 쌓이지 않을 때

1. Actions 탭에서 루틴 실행 로그 확인
2. `GITHUB_TOKEN` 만료 여부 확인 (90일 주기)
3. `config/*.yaml` 파일의 문법 오류 확인
